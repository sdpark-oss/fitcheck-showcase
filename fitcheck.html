Problem

재고는 1개, 사이즈도 1개

리세일 의류 상품은 교환이 안 됩니다. 사이즈가 안 맞으면, 구매를 포기합니다.

1개
단일 재고

리세일 의류 상품 특성상 같은 상품의 사이즈 옵션이 없습니다.

No.1
구매 망설임 요인

패션 이커머스에서 구매를 망설이게 만드는 가장 큰 요인 중 하나는 사이즈/핏에 대한 불안.

Screen 03

입력은 3개,
시간은 5초

바텀시트가 올라오면 성별, 키, 몸무게 3개만 입력합니다. 상품의 성별 정보로 자동 선택되어 있어, 실제로는 키와 몸무게 2개만 입력하면 됩니다.

페이지 이탈 없이, 바텀시트 안에서 모든 것이 완료됩니다. 스크롤 체이닝도 방지하여 배경 페이지가 따라 움직이지 않습니다.

Screen 04

판정은 한눈에,
근거는 그 아래에

AI가 상품 실측 데이터와 고객 체형을 비교 분석하여, 5단계 핏 판정을 내립니다. 종합 뱃지로 결론을 먼저 보여주고, 게이지와 MD 코멘트로 근거를 제공합니다.

MD의 한마디 — 역할

5단계 판정이 "맞는지 안 맞는지"를 알려준다면, MD의 한마디는 "그래서 이 옷을 어떻게 입으면 좋을지"를 자연어로 전달합니다.

• 최종 판정을 부정하거나 뒤집지 않음
• 핏을 취향의 언어로 연결 (단정한 인상, 편안한 실루엣 등)
• 과장 추천("강추", "꼭 소장하세요") 및 근거 없는 안심 표현 금지
• 사이즈 업/다운 제안 금지 — 현재 상품 기준의 착용감만 설명
5단계 판정 색상
타이트한 편 게이지 10%
약간 타이트한 편 게이지 30%
잘 맞는 편 게이지 50%
약간 여유 있는 편 게이지 70%
크게 느껴질 수 있음 게이지 90%
빨강 사용 금지 — 타이트 판정도 주황으로 통일하여 "사지 마" 시그널을 회피합니다.

Screen 05

바텀시트를 닫아도,
판정은 남아 있다

결과를 확인한 뒤 바텀시트를 닫으면, 미니 CTA가 판정 결과를 반영한 상태로 변환됩니다.

판정 색상 배경과 텍스트로 "확인 완료" 상태를 시각적으로 전달합니다. 다시 클릭하면 바텀시트가 재오픈됩니다.

판정별 색상이 미니 CTA에도 연동됩니다. 타이트 → 주황, 잘 맞음 → 초록, 여유 → 파랑, 큼 → 인디고.
Design System

불안을 줄이는 디자인 규칙

💡
Anxiety → Confidence

사이즈 불안을 핏 확신으로 전환하여 구매전환율을 높입니다.

📱
Zero Page Transition

바텀시트로 페이지 이탈 없이 전체 플로우를 완료합니다.

🚫
빨강 금지

타이트 판정도 주황으로. 빨강은 "구매 경고" 시그널이므로 사용하지 않습니다.

👕
사이즈 대안 금지

리세일 의류 특성상 1개 사이즈만 존재. 다른 사이즈를 제안하지 않습니다.

🛡️
Disclaimer → Reassurance

"정확하지 않을 수 있습니다" 대신 "최초 1회 무료 반품"으로 안심을 줍니다.

Beta

구성원이 직접 자기 옷의 실측을 입력하고 AI 판정 정확도를 검증할 수 있는 테스트 도구입니다.

1
내 정보 — 성별, 키, 몸무게 입력
2
옷 정보 — 복종 선택(일반/골프 8종) + 실측 직접 입력
3
실제 착용 느낌 — AI 결과 전에 본인 체감 판정을 먼저 선택 (편향 방지)
본인의 체감 판정을 먼저 기록한 후 AI 판정을 비교합니다.
이를 통해 AI가 실제 착용감을 얼마나 정확히 예측하는지 편향 없이 검증할 수 있습니다.
